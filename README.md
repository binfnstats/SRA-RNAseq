### SRA-RNAseq 
https://emojipedia.org/emoji/%F0%9F%91%89/ Include pipelines for running Pre-processing methods such as Trimming, quality control and pseudoalignment using Trimmomatic, fastQC and Kallisto tools respectively.
All the pipelines are written 
using the https://github.com/openwdl/wdl and can be executed using 
https://github.com/broadinstitute/cromwell and https://www.docker.com/.
